### Installation
#### 1. Set AWS credentials in `env.json`
```json
    {
    "aws_access_key_id": "<Access Key ID>",
    "aws_secret_access_key": "<Secret Access Key>",
    "region": "<region>"
}
```
#### 2. Install dependencies
```bash
pip3 install -r requirements.txt
```
#### 3. Run the flask application
```bash
flask --app app run
```
